# Portfolio (Particle-Wave Background)

This is a clean, single-page portfolio with an interactive canvas background.

## Run locally
- Just open `index.html` in a browser.

## Deploy (quick)
### GitHub Pages
1. Create a GitHub repo, upload these files.
2. Repo Settings → Pages → Deploy from branch.

### Vercel / Netlify
- Import the repo and deploy as a static site.

## Customize
- Edit text in `index.html` (Your Name, links, projects).
- Background settings in `script.js` (`SPACING`, wave params).
- Styling in `style.css`.
